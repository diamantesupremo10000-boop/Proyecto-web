# Sitio Web Full-Stack de Contenido

Proyecto web full-stack simple, accesible y estable, listo para desplegar en Render.

## Tecnologías
- Node.js
- Express
- HTML, CSS, JavaScript Vanilla

## Ejecución local
npm install
npm start

## Endpoint
GET /content?lang=es|en

## Despliegue
- Subir a GitHub
- Crear Web Service en Render
- Build: npm install
- Start: npm start
